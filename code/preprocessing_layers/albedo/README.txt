These downloads represent data from the following dataset:
https://www.earthdata.nasa.gov/data/catalog/lpcloud-eco-l2t-stars-002

Data was searched for manually in NASA's earth data search.
A bounding box was manally selected around London and a filter for a temporal resolution of 
2023-0701 to 2023-08-31 was selected to download. This was due to the fact we
 were looking at summer month and had an LULC layer for 2023
